Automatización de Reportes

Flujo automatizado con Power Platform que extrae datos de páginas web y genera dashboards dinámicos.

Anterior Siguiente

Descripción del Proyecto

Este proyecto consiste en un flujo desarrollado con Power Automate Desktop que visita automáticamente sitios web de líneas navieras, resuelve CAPTCHA, extrae fechas de llegada (ETD/ETA), y las guarda en un archivo de Excel en SharePoint. Luego, Power BI toma esos datos y genera visualizaciones en tiempo real. El flujo también automatiza el envío de reportes filtrados por región a los vendedores.

Para el reconocimiento de texto en imágenes se utilizó Python con OCR, combinando procesos de scraping, automatización y análisis visual.
